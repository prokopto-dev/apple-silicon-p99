# P99 on Apple Silicon

Run **[Project 1999](https://www.project1999.com/)** — the classic EverQuest
emulator — natively-ish on a modern Mac, Apple Silicon or Intel, using an
entirely free and open-source stack. No CrossOver license, no virtual machine,
no Windows install. You bring your own EverQuest Titanium files; everything
else is set up for you.

[**Download P99 Installer**](https://github.com/prokopto-dev/apple-silicon-p99/releases/latest){ .md-button .md-button--primary }

!!! note "First launch"
    The app is not notarized (no paid Apple Developer ID), so the first open
    needs **right-click → Open** to get past Gatekeeper.

Prefer the terminal? The same install is available as plain shell scripts —
see the [README](https://github.com/prokopto-dev/apple-silicon-p99#readme)
for the terminal path, system requirements, and building from source.
